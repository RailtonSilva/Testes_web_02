🧪 Testes de Login com Cypress

Este projeto contém testes automatizados de login utilizando o Cypress na aplicação ServeRest.

✅ Cenários testados:
Login com sucesso
Acessa a página de login
Insere credenciais válidas
Realiza o login
Adiciona um produto à lista de compras
Valida se o produto foi adicionado corretamente
Login com erro
Acessa a página de login
Insere credenciais inválidas
Tenta realizar o login
Verifica a exibição da mensagem de erro
🚀 Tecnologias utilizadas:
Cypress
JavaScript
📌 Objetivo:

Garantir que o fluxo de autenticação e ações básicas do usuário estejam funcionando corretamente.
